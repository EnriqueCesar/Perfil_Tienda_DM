# Perfil RD · DM · Tienda 2.0

Proyecto HTML listo para GitHub Pages.

## Archivos principales
- `index.html`: dashboard ejecutivo.
- `style.css`: estilo Starbucks ejecutivo.
- `app.js`: lógica de filtros, vistas, gráficos y PDF.
- `data.js`: dataset generado desde `Base_PerfilTiendaYTD(3).xlsx`.
- `Store_Master_Audit.csv`: auditoría rápida de cobertura por tienda.

## Fuente maestra
La fuente oficial para nombre final de tienda es **Directorio**.
Se limpian tiendas con comentario **Cierre Definitivo** en Store_Master.
